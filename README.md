## Getting Started
1. Clone the repo:
   ```bash
   git clone https://github.com/zakaahmed1/codeoff-aug25-teamvinayak.git
   cd codeoff-aug25-teamvinayak
   ```

2. Install dependencies:

  ```bash
  pip install -r requirements.txt
  ```

3. Run the project:
  ```bash
  python src/main.py
  ```
